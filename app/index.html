Hey! Hope you like my solution.

I focused on the visual aspect: you asked for a identical calculator, and you got it! I could have done this without using images, but only a CSS only solution would not be visually identical to the real calculator.

Some points I would like to make about best practices:

  • CSS is following BEM (block__element--modifier) pattern;
  • Even though I used images, all buttons and numbers are in ONE SPRITE;
  • The text inside the buttons is HTML;
  • I used only vanilla JavaScript. No jQuery here;
  • The build of this application was output by GULP. I only minified and concatenated the dependencies, but not the application code, so you can read it. You can build it yourself if you have NodeJS and npm:
    • 1. npm install (download dependencies);
    • 2. npm run calculator (build and auto-start browser with the calculator).
